# 001 Offset Capability Lab

オフセットと標準偏差を調整し、正負に移動した2つの分布のヒストグラムと工程能力指数（Cp・Cpk）を比較するブラウザアプリです。

## 主な機能

- Offset、標準偏差、Mean、USL、LSLの調整
- 10,000件の正規分布データをブラウザ内で生成
- Offset +、Offset −、全体のCp・Cpk・平均・標準偏差を比較
- Canvasによる高DPI対応ヒストグラム描画
- 再サンプリングと初期値へのリセット
- PC・スマートフォン対応

## 使い方

1. Offset ValueとStandard Deviationをスライダーで調整します。
2. 必要に応じてMean、USL、LSLを入力します。
3. 指標カードとヒストグラムの変化を確認します。
4. 同じ条件で乱数を生成し直す場合は「再サンプリング」を押します。

計算と描画はすべてブラウザ内で行い、外部へデータを送信しません。

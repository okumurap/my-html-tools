# でんしゃを走らせよう！

画面をタップして、駅から踏切とトンネルを越えて次の駅まで電車を走らせるミニゲームです。

## 使い方

1. 画面をタップするか、EnterキーまたはSpaceキーを押すと電車が出発します。
2. 走行中にもう一度タップすると少しスピードアップします。
3. 踏切とトンネルを通過して次の駅に着いたら、もう一度タップすると次の旅が始まります。
4. 右上のボタンで音のON/OFFと全画面表示を切り替えられます。

## 特徴

- PC・スマートフォン対応
- 外部通信なし
- Canvasで描く駅、踏切、トンネル、電車のアニメーション
- Web Audio APIで効果音を生成（音はOFFに切り替え可能）
- キーボード操作、画面読み上げ用ステータスに対応
- Vanilla JavaScriptのみで実装

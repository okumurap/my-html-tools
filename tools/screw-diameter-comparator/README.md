# 003 スクリュ径A/B比較ツール

射出成形機のスクリュ径A/Bについて、断面積、外周長、最大周速、理論体積流量、理論充填体積、必要ストロークを比較するブラウザツールです。

## 入力項目

- スクリュ径A・B
- 射出速度
- スクリュ回転速度
- スクリュ移動量
- 目標充填体積

## 計算内容

- 断面積: `π × D² ÷ 4`
- 外周長: `π × D`
- 最大周速: `π × D × rpm ÷ 60`
- 理論体積流量: `断面積 × 射出速度`
- 理論充填体積: `断面積 × スクリュ移動量`
- 必要ストローク: `目標充填体積 ÷ 断面積`
- ストローク径比: `スクリュ移動量 ÷ スクリュ径`

内部計算では小数を保持し、実値は整数へ四捨五入して3桁区切りで表示します。B/A倍率は小数第2位、増減率は小数第1位、ストローク径比は小数第2位まで表示します。

計算はすべてブラウザ内で行い、外部通信は行いません。

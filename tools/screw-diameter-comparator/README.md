# 003 射出成形機｜スクリュ径 A/B比較

射出成形機のスクリュ径A/Bについて、射出速度、回転数、概算移動量、射出体積を比較・換算するブラウザツールです。

## 主な機能

- スクリュ径を `φ24 / φ28 / φ32 / φ36 / φ45 mm` から選択
- A/Bそれぞれの射出速度、回転数、概算移動量を数値またはスライダーで入力
- 射出速度を編集すると、同じ体積流量になる反対側の速度を換算
- 回転数を編集すると、同じスクリュ周速になる反対側の回転数を換算
- 概算移動量と共通射出体積を双方向に換算
- 断面積、体積流量、スクリュ周速、概算射出時間を表示

## 初期条件

- 条件A: `φ28 mm`、射出速度 `50 mm/s`、回転数 `150 rpm`
- 条件B: `φ32 mm`
- 射出体積: 条件Aの概算移動量 `50 mm` に相当する体積

## 換算式

- 断面積: `π × D² ÷ 4`
- 体積流量: `断面積 × 射出速度`
- スクリュ周速: `π × D × rpm`
- 概算移動量: `射出体積 ÷ 断面積`
- 概算射出時間: `概算移動量 ÷ 射出速度`

計算はすべてブラウザ内で行い、外部通信は行いません。

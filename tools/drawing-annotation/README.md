# 002 図面注釈ツール

PNGまたはJPEGの画像・図面へ、ペン、直線、矢印、四角、楕円、連番ラベルを書き込み、PNGとしてコピーまたは保存できるブラウザアプリです。

## 主な機能

- ファイル選択、クリップボード貼り付け、ドラッグ＆ドロップによる画像読み込み
- ペン、消しゴム、直線、矢印、四角、楕円、数字ラベル
- 選択ツールによる注釈の移動・変形・削除
- 元に戻す、やり直す
- 全体表示、ホイール・ボタン・キーボードによる拡大縮小
- PNGのクリップボードコピーと保存
- PC、スマートフォン、ペン入力対応

## 使い方

1. 「画像を開く」からPNGまたはJPEGを選びます。PCでは画像の貼り付けやドラッグ＆ドロップも利用できます。
2. ツール、色、太さを選び、図面上で描画します。
3. 注釈を直す場合は「選択」を選び、移動またはハンドル操作を行います。
4. 完成した画像を「PNGコピー」または「PNG保存」で出力します。

画像と注釈はすべてブラウザ内で処理され、外部へ送信されません。画像は30MB以下、合計2,400万画素以下に対応します。

# 005 円筒度簡易ビューア 複数CSV比較版

複数のXYZ点群CSVを円筒座標へそろえ、測定データごとの形状やデータ間の差・ばらつきを比較するブラウザツールです。

## 主な機能

- CSV・TSV・TXT形式のXYZ点群を複数読み込み
- 円筒軸を自動推定、またはX・Y・Z軸から指定
- 測定点数や位置が異なるデータを共通グリッドへ補間
- 個別形状、平均、基準データとの差分、標準偏差、最大差、有効データ数を表示
- 3Dビュー、円筒展開マップ、軸方向プロファイルで可視化
- 比較グリッドと評価指標をCSVで出力
- サンプルデータを内蔵

## CSVの基本形式

X・Y・Z座標の3列が必要です。列名を認識できない場合は先頭3列をX・Y・Zとして読み込みます。任意で `section` と `point` 列を指定できます。

すべての解析はブラウザ内で行い、読み込んだデータを外部へ送信しません。本ツールの円筒度は最小二乗近似による簡易評価であり、正式な幾何公差判定には使用できません。
